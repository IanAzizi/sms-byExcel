# 📅 Check Reminder v2

سیستم یادآوری چک برای مستأجران مجتمع تجاری "بندرعباس مال"

این پروژه با هدف خواندن لیست چک‌های اجاره از فایل Excel و ارسال پیامک یادآوری به مستأجران توسعه داده شده است. نسخه دوم قابلیت‌های جدیدی را برای ارسال پیامک‌های یادآوری در بازه‌های زمانی مختلف پیش از موعد چک ارائه می‌دهد.

---

## ✅ امکانات نسخه دوم

- خواندن فایل Excel شامل اطلاعات چک‌ها
- تبدیل تاریخ شمسی به میلادی به صورت خودکار
- ارسال پیامک یادآوری در:
  - 10 روز مانده به سررسید
  - 5 روز مانده به سررسید
  - 3 روز مانده به سررسید
- ارسال پیامک هشدار پس از موعد چک
- حذف فایل Excel پس از پردازش
- لاگ کامل همه عملیات

---

## 🛠️ نصب و اجرا

### 1. کلون کردن پروژه


git clone https://github.com/your-username/check-reminder.git
cd check-reminder
2. نصب پکیج‌ها

npm install
3. ساخت فایل .env
در ریشه پروژه یک فایل .env بساز و مقادیر زیر را قرار بده:

env

SMS_API_URL=https://your-sms-api.com/endpoint
SMS_API_KEY=your_api_key
4. اجرای برنامه

node app.js
📤 ارسال فایل Excel
پس از اجرای برنامه، با استفاده از POST /upload می‌تونی فایل اکسل چک‌ها را آپلود کنی. پس از آپلود، پیامک‌ها به صورت خودکار برای موارد واجد شرایط ارسال می‌شن.

📁 فرمت فایل Excel
ستون‌های مورد نیاز:

شماره چک	نوع قرارداد	طبقه	واحد	موضوع	ماهیت	سال سررسید	تاریخ سر رسید	طرف مقابل	مبلغ	نام بانک	شماره تماس

تاریخ سررسید باید به صورت شمسی و با فرمت YYYY/MM/DD وارد شود.

💬 مثال پیامک‌ها
10 روز مانده:\
مستاجر گرامی، فقط ۱۰ روز تا تاریخ سررسید چک شماره 1234 مربوط به اجاره مغازه شماره 102 در مجتمع بندرعباس مال باقی مانده است. لطفاً برای جلوگیری از مشکل، وجه چک را تا تاریخ ۱۴۰۴.۰۵.۲۸ پرداخت نمایید.
پس از موعد:

❗️مستاجر گرامی، ضمن اعلام عدم وصول چک سریال شماره 1234 مربوط به اجاره مغازه شماره 102 در مجتمع بندرعباس مال، لطفاً برای جلوگیری از اقدام قضایی، وجه چک را حداکثر تا تاریخ ۱۴۰۴.۰۵.۲۸ کارسازی و مراتب را به مدیریت مالی مجتمع اعلام نمایید.
